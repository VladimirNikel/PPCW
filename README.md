### Запуск проекта
Необходимо перейти в директорию с проектом
```bash
cd laba1/project
```

Деплой выполняется командой
```bash
make deploy
```
Дальше в браузере достаточно открыть http://127.0.0.1:8080/, чтобы увидеть сайт

Для завершения работы с проектом, достаточно выполнить
```bash
make delete
```
